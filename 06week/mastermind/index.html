Mastermind

aka Code Breaker

WTF is Mastermind?

A game to summon the Mastermind in you and guess the four digit code.

How to play:

  • Enter your four (4) digit guess in the yellow box.
  • Your guess should at least be four (4) numbers.
  • Below the yellow box a message in appear in white letters. Read it.
  • You have ten (10) tries to guess the code!
  • Read the Legened to understand how to read your result.
  • The "$$$$" has the code hidden and will reveal itself once either you crack the code or fail trying.

Result Symbol Legend

  • $ means a number is in the correct position.
  • X means a number is not part of the code.
  • ~ means the number is part of the code, but not in the right spot - nor how many times the number exists in the code.
